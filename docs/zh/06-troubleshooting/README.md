## 🔍 **常见问题**

### 1. 服务无法启动

**问题**: 端口被占用

**解决**:
```bash
# 检查端口占用
lsof -i :8080

# 修改端口
export PORT=8081
```

### 2. 代理失败

**问题**: 上游服务不可用

**解决**:
- 检查网络连接
- 检查上游服务状态
- 查看日志文件

### 3. 性能问题

**问题**: 响应慢

**解决**:
- 查看 [性能优化](../05-advanced/performance.md)
- 检查资源使用情况
- 调整并发限制
