## 🏗️ **架构概览**

CDNProxy 采用模块化设计：

- **代理层**: CDN代理和API代理
- **缓存层**: 磁盘缓存和内存缓存
- **管理层**: 管理面板和配置管理
- **监控层**: 性能监控和健康检查
