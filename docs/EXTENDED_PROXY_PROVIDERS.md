# 扩展住宅IP代理提供者列表

基于市场调研，以下是更多可用的住宅IP代理服务商：

- **IP数量**: 1.5亿+
- **覆盖地区**: 全球所有国家

- **价格**: $8.4/GB

#### **Smartproxy** (已支持)
- **IP数量**: 1.15亿+

- **协议支持**: HTTP(S)
- **价格**: $7/GB
- **特点**: 价格便宜，易于使用
